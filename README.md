Project Overview: 

This project features a fully functional, dynamic Power BI dashboard designed to analyze mobile phone sales data. The objective was to transform raw sales records into actionable insights, helping stakeholders understand brand performance, customer demographics, and regional sales trends.The data underwent a rigorous ETL process using Power Query to ensure accuracy and consistency before being visualized.
